Supporting note

Method

The projects start with practical judgment problems, then move through artifact building, boundary review, and public inspection.

Working pattern

  • Frame the judgment problem in plain terms.
  • Build something concrete enough to test and explain.
  • Review claims, limits, source posture, and use conditions.
  • Inspect how the work reads in context.
